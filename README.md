# AI_kinolights

키노라이츠 스타일 OTT 통합 검색/추천 메인 페이지 예시입니다.

## 로컬 실행

```bash
npm install
npm run dev
```

## GitHub Pages 배포

이 저장소에는 `.github/workflows/deploy-pages.yml` 워크플로우가 포함되어 있으며, `main` 브랜치로 push 하면 자동으로 정적 빌드(`dist`)가 GitHub Pages로 배포됩니다.

- 예상 URL 형식: `https://<GitHub-username>.github.io/AI_kinolights/`
- 실제 URL은 GitHub 저장소의 **Settings → Pages** 또는 워크플로우 실행 결과에서 확인할 수 있습니다.
